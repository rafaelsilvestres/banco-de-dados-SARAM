# Banco de Dados SARAM

O presente relatório apresenta a elaboração do banco de dados SARAM (Sistema Acadêmico de Ração Animal e Manejo). Tal projeto foi realizado em parceria com a empresa “Saúde Animal”, um pet shop que possui casa de ração, banho e tosa e veterinário. Em nossa parceria, decidimos abordar somente a parte da casa de ração, usando desse contexto e da natureza desse negócio para elaborarmos o nosso projeto em banco de dados. Nessa perspectiva, usamos as especificações do parceiro a fim de construir o modelo relacional, sendo geradas seis entidades e cinco relacionamentos. Por conseguinte, tais entidade e relacionamentos conceituais se transformaram em onze tabelas no modelo lógico, as quais foram implementadas no modelo físico por meio do sgbd Postgresql.
